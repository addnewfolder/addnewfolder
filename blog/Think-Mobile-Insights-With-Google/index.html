Think Mobile With Google：360度行動時代洞察

近日，我有幸參加了 Google 在台北舉辦的「Think Mobile With Google in 360 degree」活動，活動中多位業界專家分享了他們對於行動時代的見解與策略。以下是我整理的重點內容，供大家參考。

一、李飛（Lee-feng Chien）：行動無處不在

李飛首先指出，「Mobile Everywhere」已經成為現實，僅僅八年的時間，人類就從傳統的互聯網時代進入了行動互聯網時代，成為了「低頭族」。他強調了以下幾個關鍵點：

  • 行動購物崛起：美國史上首次網路銷售額超過實體銷售額，其中行動購物佔36%。
  • Mobile First/Only 的年代：預計年底，手機搜尋量將超過 PC。
  • 全球行動支付的發展：即使在肯亞這樣的國家，透過 2G 網路也能實現支付功能。
  • 行動廣告收入增長：目前行動廣告收入仍有巨大的增長空間。
  • 台灣市場的潛力：台灣在 Google Play 上市規模全球第五。
  • 行動影音的增長：在手機上觀看超過 20 分鐘的影片用戶越來越多。
  • 未來展望：預言明年將進入「車聯網」（Internet of Cars）時代，屏幕數量將增至七個，帶來更多機遇。

二、周尹（Eric Chou）：Google Analytics 的應用

Eric Chou 從 Google Analytics（GA）的角度，講解了如何透過數據了解使用者，發現程式流程和介面設計上的缺陷，進而提高轉換率。 link

1. 了解你的使用者

  • 來源報表：分析使用者的來路，了解他們從哪裡進入網站或應用。
  • 同類群組分析（Cohort Analysis）：透過橫軸查看回訪率，了解用戶留存情況。
  • 客層和興趣報表：深入了解用戶的背景和興趣，進行精準行銷。

2. 積極查看報表案例

  • 平台裝置和地區分析：例如，發現大部分用戶由手機進入，且集中在某地區的某款平板，可能是該地區的裝置預裝了你的服務。
  • 當機/例外狀況報表：針對 App，可以直接發送 Email 通知相關人員。
  • 行為流程報表：注意使用者的離開率（Drop Rate），如需追蹤訂房結果，可以做跨網頁的追蹤，避免不合理的行為流程。
  • 事件追蹤：對於 App 上的按鈕擺放，應該能夠追蹤使用效果，例如滑動產品介紹頁等。

3. 事件的最佳實踐

  • 精簡事件數量：由於事件有上限，沒有用的事件就不要發送。
  • 統一命名原則：設計者需要了解報表使用者的需求，並採用規定的命名原則（區分大小寫）。
  • 觀察 UI 設計效果：透過事件偵測，了解用戶是否與 UI 進行了互動。

4. 轉換率與業務結合

  • 設定目標：找出最重要的可量化指標，如轉換率，並與業務目標相結合。
  • 了解用戶行為：例如，如何知道用戶是否已登入？可以透過 GA 偵測 Cookie 等方式。

5. 電子商務功能強化

  • 交易和結帳細節：將這些資訊傳送到 GA，可以了解完整的流程和示意圖。
  • 內容實驗：設定目標，進行 A/B 測試，優化用戶體驗。

三、Birdy：廣告業主如何投放廣告？

Birdy 主要講述了廣告業主在行動時代的投放策略，以及 Google Display Network（GDN）的應用。

1. 程序化購買（Programmatic Buying）

  • 概念：利用技術，在對的時間，向對的人，傳遞對的內容。
  • 實現方式：依據性別、興趣等投放廣告，採用實時競價（Real-Time Bidding）。

2. 廣告業主的兩種需求

  • 品牌行銷（Branding）：提升知名度，但可能不直接在官網進行購買，例如汽車、IBM 等。
  • 績效行銷（Performance）：直接帶來轉換，如電商購物。

3. 定義行銷目標

  • Branding：設定投放期間、預算、目標受眾（TA）、目標（知名度或網站造訪度）。
  • Performance：設定投放期間、預算、目標受眾（TA）、目標（例如申請信用卡）。

4. 廣告計費模式

  • CPM（Cost Per Mille）：按千次曝光計費，適合提升品牌知名度。
  • CPE/CPV（Cost Per Engagement/View）：按互動或觀看次數計費，適合提升互動率。
  • CPC/CPA（Cost Per Click/Action）：按點擊或行動計費，適合導流到網站，提高轉換率。

5. 對出版商/應用開發者的建議

  • 適應市場趨勢，重視 UX：提供優質的用戶體驗。
  • 提供優質內容：持續吸引和留住用戶。
  • 細分網頁類別：如電影、運動等，方便精準投放。
  • 保持冷靜並優化：根據不同季節開放更多版位，靈活調整策略。

四、行動網頁：速度為王

在行動時代，網頁速度對於用戶體驗和營收都有著至關重要的影響。

1. 速度影響營收

  • 用戶訪問設備：手機訪問佔約 60%，平板佔 10%，其餘為 PC。
  • 速度與體驗：Google 發現，每慢 200 毫秒，體驗就會下降 0.3%，帶來巨大的損失。
  • 用戶耐心：如果網站加載超過 10 秒，49% 的用戶會放棄使用，其中一半不會再回來。

2. 速度與指標

  • 理想的載入時間：3~6 秒以下為佳，但不能為了速度犧牲內容。
  • Speed Index：目標接近 1000，Google PageSpeed Insights（PSI）目標為 85，載入時間 8 秒。
  • 實例：ESPN 透過優化，將重要資訊的載入時間從超過 20 秒縮短到更短。

3. 提升用戶體驗

  • 關注內容頁：不僅主頁需要快速加載，內容頁也同樣重要。
  • 速度與體驗相關：優化速度有助於提升用戶滿意度和網站留存率。

五、Cloe Tai：五大行動跨螢幕黃金營利策略

Cloe Tai 分享了在多螢幕時代，如何優化廣告策略，提升營收。

1. 響應式廣告設計

  • 多螢廣告的重要性：21% 的用戶已經使用手機瀏覽網頁，因此廣告也需要響應式設計，以適應不同裝置。

2. 提高廣告可見性

  • Viewability Rate：指有 50% 的像素被用戶看到的廣告。
  • 內容中插入廣告：效果也不錯，但每篇文章最好只放置一個，並保持適當間距。

3. 優化廣告尺寸和位置

  • 關鍵時期調整：在廣告預算較高的第四季度，觀察並優化廣告的尺寸和位置。
  • 推薦尺寸：320x100、300x250、336x280 等，這些尺寸都是經過實驗驗證的。

4. 增加廣告覆蓋率

  • 長文章多廣告：如果文章較長，可以申請放置兩個以上的廣告（預設為一個）。
  • 文字廣告的效果：文字形式的廣告也有不錯的表現。

5. 豐富廣告形式

  • Anchor Ads：置底廣告，持續出現在用戶視野中。
  • Vignette Ads：滑動內容時出現的全屏廣告，需要申請使用。
  • Native Ads：在文章中間插入與內容風格相似的廣告，可自訂底色和大小，CTR 表現優秀。

六、Programmatic Buy：程序化購買的未來

1. Programmatic Direct 的發展

  • 市場趨勢：中國的程序化市場將越來越大，預計 2017 年達到 9%，美國達到 30%。
  • 傳統模式的轉變：過去媒體直接與廣告商談判版位和時間，現在則轉向程序化購買。

2. 雙方需求的平衡

  • 廣告主的期望：希望以較高的 CPM 獲得更好的效果。
  • 出版商的期望：希望版面能有更大的彈性運用，提升收益。

七、得獎者座談會：業界分享

在座談會上，多位業界代表分享了他們對於行動時代的見解。

1. Mobile 對網站的影響

  • Justin：注意到手機版流量的重要性，建議建立專門的團隊經營和優化手機體驗和動線。
  • Pixnet：手機需要滿足用戶的當下需求，精準掌握內容的提供。
  • Mobile01：雖然目前 PC 流量仍然較高，但也投入大量時間優化手機版，內容和速度至關重要，需要在廣告和用戶體驗之間取得平衡。

2. Mobile Web 還是 App？

  • Pixnet：由於大部分訪客是透過搜尋或分享連結進入，因此更注重行動網頁而非 App。綜合型內容不適合做成 App，除非針對特定使用情境進行內容重組和規劃，如美食或旅遊。

3. 借力社群和分享，實現病毒式行銷

  • Justin：要求編輯構思好的標題和首張圖片，滿足 80% 的分享需求，並與編輯多溝通，創作讓人想分享的文章。

4. 持續產出優質內容的動力

  • Mobile01：重視原創內容，速度次之。透過與 Google 合作，進行廣告實驗，保持編輯的獨立性。

5. 對未來網路廣告和市場的建議

  • Justin：注重原生廣告和影音廣告，拿到預算後，製作不讓用戶反感的優質廣告。
  • Mobile01：理想上希望沒有廣告，但需要在生存和用戶體驗之間取得平衡，希望大家都能做對的事情。
  • Pixnet：專注於內容和正確的事情，為用戶提供價值。

總結

此次活動讓我深刻感受到行動時代帶來的挑戰與機遇。無論是行動購物的崛起、行動廣告的增長，還是用戶體驗的優化，都需要我們以開放的心態去學習和適應。通過深入了解用戶、優化內容與廣告策略，以及積極採用新技術，我們才能在激烈的市場競爭中脫穎而出。
